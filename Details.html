Descriere

Aceasta aplicatie Web mediaza schimbul de carti intre utilizatorii care se afla aproximativ in aceeasi locatie.

BooX este o aplicatie cu ajutorul careia se pot vizualiza detalii despre cartile oferite de utilizatorii aflati in zona geografica proxima, cat si despre carti care nu sunt oferite de utilizatori, dar se doresc spre a fi luate.

Aplicatia web este reprezentata de un site in care utilizatorul are posibilitatea sa adauge carti pe care doreste sa le ofere la schimb, cat sa si primeasca notificari legate de un posibil partener de trade, specificatii care constau in: similitudinea tipurilor de carti, zona geografica, cartile adaugate de sine ca si wanted.

HTML,CSS, Javascript - Front End

SQL - DataBase
